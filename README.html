GmailArchive

  • What is it?

    • A macro for Gmail written in Google Apps Script.

  • What is it for?

    • Those who, for whatever reason, prefer to save their messages in Gdrive instead of the Gmail folders.

      • Save protocols, reports or memos directly from Gmail to a private/public folder all in one click.

      • Organize your projects strictly by date and time for easy future reference.

  • How does it work?

    • Every minute the code searches your Gmail threads for messages that you want save to Gdrive. It then converts your threads into the PDF or GDOC format and saves them to your specified folder of your Gdrive.

  • How do I use it?

    • Create the following Gmail labels:

      • "RootDoc"

      • "RootPDF"

      • "ReferenceDOC"

      • "ReferencePDF"

    • Open Google Apps Script (https:script.google.com/macros) and create a "blank project".

    • Copy and paste the code, then hit save (name it whatever you want).

    • Run "createTriggers".

    • It will ask you to authorize, after you authorize run "createTriggers" again just to be sure that it worked.
